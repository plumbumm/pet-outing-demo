学习
玩法介绍

带宠物出门上课！有文化、科学、艺术、体育四门科目可以选。

每次上课消耗 10点体力，获得1个学分，Lv.0课程免费，后续课程需要缴纳学费（金币）。学完之后宠物会变得更强——战力属性会提升，还有机会学到新的技能。

每门科目有独立的等级，等级没有上限，可以一直提升变强。

怎么升级

每升一级需要上一定次数的课。等级越高，需要的次数越多：

升级所需次数 = 当前等级 + 2

举例：Lv.0升到Lv.1 需要上2次课
Lv.5升到Lv.6 需要上7次课
学费

刚开始学是免费的，后面随等级提高学费会涨，最多只要300金一次：

Lv.0免费

不同的科目有什么用？

每门科目会提升不同的战力属性，选什么科目决定了你的宠物擅长什么：

科目主要提升次要提升
文化脑力灵巧
科学体魄脑力
艺术灵巧脑力

小贴士：属性会影响PK胜负哦！
· 想让宠物变成脑力型 → 多学文化和艺术
· 想让宠物变成灵巧型 → 多学艺术和体育
· 想让宠物变成体魄型 → 多学科学和体育
学习能获得技能吗

能！每次学完都有机会习得一个和当前科目相关的技能。学得越多，获得技能的机会越多。

打工
玩法介绍

带宠物出门打工赚钱！有6个工种可以选，每次打工消耗 10点体力，不花金币。

每个工种有独立的等级，等级越高收入越多，职业头衔也会跟着变。打工也有机会获得技能。

能赚多少钱？

升级规则和学习相同（升级所需次数 = 当前等级 + 2）。收入随等级增长，最高1500金一次：

每次收入 = 100 + 等级 x 50

举例：Lv.0赚100金 / Lv.5赚350金 / Lv.10赚600金 / Lv.20赚1100金
最高不超过1500金/次
雇佣好友宠物打工

你可以雇佣一只好友的宠物来帮忙，两个人一起干活效率更高：

  • 雇佣后，这次打工的总工资翻倍
  • 但要把其中 1/4 分给好友作为酬金
  • 也就是说：你实际到手是平时的1.5倍，好友躺赚你工资的一半

注意事项：

  • 好友的宠物被雇走期间不能自己出门（好友可以随时喊回来）
  • 每只宠物每天只能被雇佣1次
冒险
玩法介绍

带宠物出门探索，每次出门需要购买门票（200一张）。冒险途中会随机遇到各种事件——可能赚到钱，可能遇见别的宠物，还可能发生一些意想不到的事哦～

PK
怎么打

PK靠三个属性决胜负：体魄、灵巧、脑力。三者之间存在克制关系：

  • 体魄 克制 灵巧——花里胡哨没有用
  • 灵巧 克制 脑力——天下武功，唯快不破
  • 脑力 克制 体魄——你有蛮力，我有策略

你的宠物属于什么类型，取决于三项属性里哪个最高。

战力怎么算

战力 = 三项属性之和 x 克制加成 x 随机波动

  • 克制对方：战力 x1.2
  • 被对方克制：对方战力 x1.2
  • 类型相同：正面比拼，无克制加成

每场PK还有一点随机波动，不是每次都稳赢。

玄学事件

偶尔会触发意想不到的事，可能会颠覆PK的胜负局势，概率很小，遇到算你幸运（或者倒霉）。

赢了获得什么？

赢了拿 50~80 金，宠物心情也会变好。输了只有 10 金的安慰奖，心情会稍微下降。
